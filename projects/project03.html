Project 3:

Making Your Page Mobile-Friendly: Example Site

Goal:

Make your webpage look good on both desktop and mobile devices. Learn how to use CSS to create a responsive layout!

Instructions:

Start with Your Project 2 Files:

  • Make a copy of your "Project2" folder and rename it to "Project3"

Update Your CSS File:

  • Open your styles.css file and add these responsive styles: 
    
/* Make the page look good on all devices */
body {
    max-width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

/* Make images responsive */
img {
    max-width: 100%;
    height: auto;
}

/* Style for mobile devices */
@media screen and (max-width: 600px) {
    body {
        padding: 10px;
    }
    
    h1 {
        font-size: 24px;
    }
    
    nav {
        text-align: center;
    }
    
    nav a {
        display: block;
        margin: 5px 0;
    }
}

Test Your Page:

  1. Open your page in Chrome
  2. Right-click anywhere on the page and select "Inspect"
  3. Click the "Toggle device toolbar" button (looks like a phone/tablet)
  4. Try different device sizes to see how your page looks

Make Adjustments:

  • If something doesn't look right on mobile:
    • Check if your images are too large
    • Make sure text is readable
    • Adjust spacing if needed

Submission:

  • Make sure your Project3 folder contains:
    • index.html file
    • styles.css file with responsive styles
    • images folder with your images
  • Compress your "Project3" folder into a zip file
  • Submit your zip file here: Submit Project 3
