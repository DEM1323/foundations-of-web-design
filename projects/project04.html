Project 4:

Adding Animations and Interactivity: Example Site

Goal:

Add some fun animations and interactive elements to your webpage using CSS and basic JavaScript!

Instructions:

Start with Your Project 3 Files:

  • Make a copy of your "Project3" folder and rename it to "Project4"

Add CSS Animations:

  • Open your styles.css file and add these animation styles: 
    
/* Add hover effect to links */
a:hover {
    color: #ff6b6b;
    transition: color 0.3s ease;
}

/* Add fade-in animation to sections */
section {
    opacity: 0;
    animation: fadeIn 1s ease forwards;
}

@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

/* Add a simple hover effect to images */
img:hover {
    transform: scale(1.05);
    transition: transform 0.3s ease;
}

Add Basic JavaScript:

  • Create a new file named "script.js" in your Project4 folder
  • Add this code to script.js: 
    
// Add a simple click counter
let clickCount = 0;

function countClicks() {
    clickCount++;
    alert('You clicked ' + clickCount + ' times!');
}

// Add this to your HTML file's body tag:
// <button onclick="countClicks()">Click me!</button>
  • Add this line to your HTML file's <head> section: 
    
<script src="script.js"></script>

Test Your Page:

  1. Run Your Project with Live Server:
    • Right-click on your "index.html" file
    • Select "Open with Live Server"
  2. Try out the animations:
    • Hover over links to see the color change
    • Hover over images to see them scale up
    • Watch sections fade in as you scroll
    • Click the button to test the counter

Submission:

  • Make sure your Project4 folder contains:
    • index.html file
    • styles.css file with animations
    • script.js file with your JavaScript code
    • images folder with your images
  • Compress your "Project4" folder into a zip file
  • Submit your zip file here: Submit Project 4
