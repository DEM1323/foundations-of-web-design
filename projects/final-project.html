Final Project:

Creating Your Portfolio Website: Example Site

Goal:

Create a beautiful portfolio website that showcases all your projects from this course!

Instructions:

Create Your Portfolio Folder:

  • Create a new folder named "Portfolio" in your workspace
  • Create these files in your Portfolio folder:
    • index.html (main page)
    • styles.css (for styling)
    • script.js (for interactivity)
    • images folder (for your images)

Set Up Your Main Page:

  • Open index.html and add this basic structure: 
    
<!DOCTYPE html>
<html>
<head>
    <title>My Portfolio</title>
    <link rel="stylesheet" href="styles.css">
    <script src="script.js"></script>
</head>
<body>
    <header>
        <h1>My Portfolio</h1>
        <nav>
            <a href="#about">About Me</a>
            <a href="#projects">My Projects</a>
            <a href="#contact">Contact</a>
        </nav>
    </header>

    <section id="about">
        <h2>About Me</h2>
        <p>Write a brief introduction about yourself here.</p>
    </section>

    <section id="projects">
        <h2>My Projects</h2>
        <div class="project-grid">
            <div class="project-card">
                <h3>Project 1</h3>
                <p>Description of your first project</p>
                <a href="project1/index.html">View Project</a>
            </div>
            <div class="project-card">
                <h3>Project 2</h3>
                <p>Description of your second project</p>
                <a href="project2/index.html">View Project</a>
            </div>
            <div class="project-card">
                <h3>Project 3</h3>
                <p>Description of your third project</p>
                <a href="project3/index.html">View Project</a>
            </div>
            <div class="project-card">
                <h3>Project 4</h3>
                <p>Description of your fourth project</p>
                <a href="project4/index.html">View Project</a>
            </div>
        </div>
    </section>

    <section id="contact">
        <h2>Contact Me</h2>
        <p>Add your contact information here.</p>
    </section>

    <footer>
        <p>© 2024 Your Name</p>
    </footer>
</body>
</html>

Add Your Styles:

  • Open styles.css and add these styles: 
    
/* Basic styles */
body {
    font-family: Arial, sans-serif;
    line-height: 1.6;
    margin: 0;
    padding: 20px;
}

/* Header styles */
header {
    text-align: center;
    padding: 20px;
    background-color: #f0f0f0;
    margin-bottom: 30px;
}

nav a {
    margin: 0 10px;
    text-decoration: none;
    color: #333;
}

/* Project grid */
.project-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
    gap: 20px;
    padding: 20px;
}

.project-card {
    border: 1px solid #ddd;
    padding: 15px;
    border-radius: 5px;
    text-align: center;
}

.project-card:hover {
    box-shadow: 0 0 10px rgba(0,0,0,0.1);
    transition: box-shadow 0.3s ease;
}

/* Responsive design */
@media screen and (max-width: 600px) {
    body {
        padding: 10px;
    }
    
    .project-grid {
        grid-template-columns: 1fr;
    }
}

Add Your Projects:

  • Copy all your project folders (Project1, Project2, Project3, Project4) into your Portfolio folder
  • Update the project descriptions in index.html to match your actual projects
  • Make sure all project links work correctly

Test Your Portfolio:

  1. Run Your Project with Live Server:
    • Right-click on your "index.html" file
    • Select "Open with Live Server"
  2. Check that everything works:
    • All navigation links should work
    • Project cards should look good
    • Page should look good on both desktop and mobile
    • All project links should work

Submission:

  • Make sure your Portfolio folder contains:
    • index.html file
    • styles.css file
    • script.js file
    • images folder
    • All your project folders (Project1, Project2, Project3, Project4)
  • Compress your "Portfolio" folder into a zip file
  • Submit your zip file here: Submit Final Project
