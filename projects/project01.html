Project 1:

Creating Your First Webpage: Example Site

Goal:

Create a simple webpage about yourself using basic HTML elements. Learn how to structure content and add links!

Instructions:

Create Your Project Folder:

  • Create a new folder named "Project1" in your workspace
  • Inside this folder, create a new file named "index.html"

Set Up Your HTML File:

  • Open your index.html file and add this basic HTML structure: 
    
<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Welcome to My Page</h1>
    
    <nav>
        <a href="#about">About Me</a>
        <a href="#interests">My Interests</a>
        <a href="#contact">Contact</a>
    </nav>

    <section id="about">
        <h2>About Me</h2>
        <p>Write a few sentences about yourself here.</p>
    </section>

    <section id="interests">
        <h2>My Interests</h2>
        <p>Share some things you like to do!</p>
    </section>

    <section id="contact">
        <h2>Contact Me</h2>
        <p>Add your email or social media links here.</p>
    </section>
</body>
</html>

Add Your Content:

  • Replace the placeholder text with your own information:
    • Change the title to something you like
    • Add your name in the h1 heading
    • Write about yourself in the About Me section
    • List your interests in the Interests section
    • Add your contact information in the Contact section

Add Some Lists:

  • Add an unordered list of your favorite things: 
    
<ul>
    <li>First favorite thing</li>
    <li>Second favorite thing</li>
    <li>Third favorite thing</li>
</ul>
  • Add an ordered list of your goals: 
    
<ol>
    <li>First goal</li>
    <li>Second goal</li>
    <li>Third goal</li>
</ol>

Add Links:

  • Add at least one link to a website you like: 
    
<a href="https://www.example.com">Visit my favorite website</a>

Viewing Your Web Page:

  1. Run Your Project with Live Server:
    • Right-click on your "index.html" file
    • Select "Open with Live Server"
    • This will open your web page in your browser
  2. Click the navigation links to make sure they work!

Submission:

  • Make sure your Project1 folder contains:
    • index.html file with your content
  • Compress your "Project1" folder into a zip file
  • Submit your zip file here: Submit Project 1
