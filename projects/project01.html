Project 1:

Creating Your First Website: Example Site

Goal:

Create a three-page website about a topic you're passionate about using basic HTML elements. Learn how to structure content and link between pages!

Instructions:

Create Your Project Folder:

  • Create a new folder named "Project1" in your workspace
  • Inside this folder, create three HTML files:
    • index.html (home page)
    • page2.html (second page)
    • page3.html (third page)

Set Up Your HTML Files:

  • Open each HTML file and add this basic HTML structure: 
    
<!DOCTYPE html>
<html>
  <head>
    <title>Your Page Title</title>
  </head>
  <body>
    <h1>Your Main Heading</h1>

    <nav>
      <a href="index.html">Home</a>
      <a href="page2.html">Page 2</a>
      <a href="page3.html">Page 3</a>
    </nav>

    <section>
      <h2>First Section</h2>
      <p>Your content here...</p>
    </section>

    <section>
      <h2>Second Section</h2>
      <ul>
        <li>List item 1</li>
        <li>List item 2</li>
      </ul>
    </section>
  </body>
</html>

Add Your Content:

  • For each page, replace the placeholder text with your own information:
    • Choose a topic you're interested in (like movies, games, sports, etc.)
    • Write about different aspects of your topic on each page
    • Add appropriate headings and sections
    • Include lists of related items

Add Lists:

  • Add an unordered list of items: 
    
<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>
  • Add an ordered list of items: 
    
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Viewing Your Website:

  1. Run Your Project with Live Server:
    • Right-click on your "index.html" file
    • Select "Open with Live Server"
    • This will open your web page in your browser
  2. Click the navigation links to make sure they work!

Submission:

  • Make sure your Project1 folder contains:
    • index.html file
    • page2.html file
    • page3.html file
  • Compress your "Project1" folder into a zip file
  • Submit your zip file here: Submit Project 1
